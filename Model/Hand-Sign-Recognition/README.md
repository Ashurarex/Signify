# Hand Sign Recognition

AI-powered hand sign recognition system using computer vision and FastAPI for real-time gesture-based communication.

---

## 🚀 Features
- Real-time hand gesture detection
- AI/ML-based recognition
- FastAPI backend
- Simple UI with action buttons
- Easy to extend with more gestures

---

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- FastAPI
- Uvicorn

---

## 🎯 Use Cases
- Assistive communication (speech-impaired users)
- Gesture-based control systems
- Emergency signaling (Help / Thirsty / Alert)
- Human-computer interaction

---

## ▶️ How to Run

```bash
# Install dependencies
python -m pip install -r requirements.txt

# Run backend
python -m uvicorn api:app --host 127.0.0.1 --port 8000
